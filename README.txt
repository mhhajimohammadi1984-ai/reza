📌 اجرای پروژه HelloFX در VS Code

1. پوشه JavaFX SDK خود را داخل فولدر lib کپی کنید 
   (یا فقط مسیر درستش رو در launch.json بنویسید).

2. در VS Code، کل پروژه HelloFX را باز کنید.

3. از منوی Run → Start Debugging (یا F5) برنامه را اجرا کنید.

✅ باید یک پنجره JavaFX با متن "سلام به JavaFX در VS Code 👋" باز شود.
